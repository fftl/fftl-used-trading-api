# Used Trading API 

## 개요

중고거래 어플리케이션의 기능들을 보고 해당 기능들을 구현해 보고 싶었습니다. 그래서 당근마켓, 번개장터 등의 어플리케이션을 보고 필요한 기능과, 필요한 데이터를 나름대로 파악해 보고 구현을 시작해 보았습니다.

## 사용 기술

- Java11
- Spring Boot 2.5
- Gradle
- Spring Data JPA
- H2
- AWS S3
- AWS EC2
- docker-compose
- jenkins
